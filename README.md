# PPiIT-Group-Project
This is our group project for the module Professional Practices in IT
## Architecture:
<img width="791" height="956" alt="Architecture diagram" src="https://github.com/user-attachments/assets/569477d4-5ec1-4e70-973d-16319af62097" />

The POS system is a client installed application that fetches information on products from a secure Amazon Web Services database. It is based in React, meaning
that it is installed on the client pc. The original goal of the application was to create an app that could be used to edit the database, however it's current
functionality is based around read-only features due to time constraints. The current implementation allows users (presumably staff) to view the items and their
relevant information in a clear and concise way.

## Installation:

	1. Extract the exe file.
	2. Run the application.
	3. Continue past the warning.
	4. Search "frontend" on your pc
	5. Click the application.

## Usage:

	1. Click the Inventory Page
	2. View the relevent item information
